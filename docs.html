BoxLANG Complete Syntax & Command Reference

Basic Syntax Rules

  • Case-insensitive keywords: b, B, box, BOX
  • No semicolons needed; one statement per line
  • Whitespace is flexible
  • Comments start with //
  • Block delimiters: " or '

Data Storage - Boxes

Boxes hold any type of data; no type declaration required.

b name John
b age 25
b pi 3.14159
b isAdult true

Variable References

Access box value using $ prefix.

b age 25
b nextAge $age [+] 1
say $age

Output Command - say

Print text or box values. Use ~ for space and : for concatenation.

say Hello~$name
say $greeting:$name

Input Command - ask

Prompt user and store input in a box.

ask What is your name? name
ask Enter your age: age

Operators

Arithmetic and comparison use brackets.

b sum $a [+] $b
b diff $a [-] $b
b product $a [*] $b
b quotient $a [/] $b
b isAdult $age [>=] 18

Conditionals - if/else

if $age [>=] 18"
  say You are an adult
else'
  say You are a minor
'
"

Loops - while

b counter 1
while $counter [<=] 5"
  say Count:~$counter
  b counter $counter [+] 1
"

Comments

// Single line comment
b age 25  // Inline comment

Complete Examples

// Basic Calculator
b num1 10
b num2 5
b sum $num1 [+] $num2
say Sum:~$sum

// User Age Checker
ask Enter your age: age
if $age [<] 13"
  say You are a child
else'
  say You are older
'
"
