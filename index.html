BoxLANG Language Reference

BoxLANG is a simple, typeless programming language where data is stored in "boxes". No type declarations needed - everything just works!

Key Features
  • ✓ Typeless - boxes hold any data
  • ✓ Bracket operators for clarity
  • ✓ Simple syntax, powerful features
  • ✓ Easy to learn and use

Basic Syntax Rules

  • Case insensitive keywords: b, B, box all work
  • No semicolons: One statement per line
  • Flexible whitespace: Spaces and tabs are flexible
  • Comments: Use // for rest-of-line comments
  • Block delimiters: Use " or ' (interchangeable)

Data Storage - Boxes

Boxes are containers that hold any type of data. They're completely typeless!

Creating/Setting Boxes

b boxName value
box boxName value    // Alternative syntax

Examples

b name John          // String
b age 25             // Integer
b pi 3.14159         // Float
b isAdult true       // Boolean
b score 100          // Number

Deleting Boxes

b boxName            // Blank box auto-deletes
Auto Types
BoxLANG automatically handles: Numbers (25, 3.14), Strings (Hello), Booleans (true, false)

Variable References

Use the $ prefix to access box values.

b age 25
b nextAge $age [+] 1    // nextAge = 26
say $age                // Outputs: 25

Recursive Resolution

Variables can reference other variables:

b a 10
b b $a              // b = 10
b c $b              // c = 10 (resolves through b to a)

Output Command - say

Prints text and variables to the console.

say Hello World
say $name
say Hello~$name
say Your age is~$age~years old

Concatenation in say

Operator Purpose Example Output
~ Join with space Hello~World Hello World
: Join without space Hello:World HelloWorld 
b greeting Hello
b name John
say $greeting~$name     // Output: "Hello John"
say $greeting:$name     // Output: "HelloJohn"

Input Command - ask

Prompts the user for input and stores it in a box.

Important
The last word is always the box name where input is stored. 
ask What is your name? name
ask Enter your age: age
say Hello~$name~you are~$age~years old

Operators

Arithmetic Operators (Bracket Only)

All math operations use bracket notation:

[+]
Addition
[-]
Subtraction
[*]
Multiplication
[/]
Division
b sum $a [+] $b
b difference $a [-] $b
b product $a [*] $b
b quotient $a [/] $b

// Complex expressions (follows order of operations)
b result 2 [+] 3 [*] 4     // result = 14 (multiply first)

Comparison Operators

Operator Comparison Example
[>] Greater than $age [>] 17
[<] Less than $age [<] 65
[=] Equal to $age [=] 25
[>=] Greater or equal $age [>=] 18
[<=] Less or equal $age [<=] 100
[|] Same sign -5 [|] -10 
b age 25
b isAdult $age [>] 17        // true
b sameSign -5 [|] -10        // true (both negative)
b diffSign -5 [|] 10         // false (different signs)

Conditionals - if/else

Execute code based on conditions.

Basic if Statement

if condition"
    // code to execute if true
"

if/else Statement

if $age [>=] 18"
    say You are an adult
else'
    say You are a minor
'
"

Nested Conditionals

b score 85
if $score [>=] 90"
    say Grade A
else'
    if $score [>=] 80"
        say Grade B
    else'
        say Grade C
    '
'
"
Delimiters
You can use either " or ' as delimiters. They can even be mixed! Each block just needs matching open/close.

Loops - while

Repeat code while a condition is true.

Basic Loop

b counter 1
while $counter [<=] 5"
    say Count:~$counter
    b counter $counter [+] 1
"

Countdown

b num 10
while $num [>] 0"
    say $num
    b num $num [-] 1
"
say Blast off!

Condition Variable

b keepGoing true
b count 0

while $keepGoing"
    say Loop iteration~$count
    b count $count [+] 1
    
    if $count [>=] 3"
        b keepGoing false
    "
"

Complete Examples

Calculator

b num1 10
b num2 5
b sum $num1 [+] $num2
b diff $num1 [-] $num2
say Sum:~$sum
say Diff:~$diff

Age Checker

ask Enter your age: age

if $age [<] 13"
    say Child
else'
    if $age [<] 20"
        say Teenager
    else'
        say Adult
    '
'
"

Countdown

b count 10

while $count [>] 0"
    say $count
    b count $count [-] 1
"

say Blast off!

Guessing Game

b secret 7
b correct false

while $correct [=] false"
    ask Guess 1-10: guess
    
    if $guess [=] $secret"
        say Correct!
        b correct true
    else'
        say Wrong!
    '
"

Multiplication Table

b number 5
b counter 1

say Table for~$number

while $counter [<=] 10"
    b result $number [*] $counter
    say $number~x~$counter~=~$result
    b counter $counter [+] 1
"

Temperature Converter

ask Enter Celsius: celsius

// Formula: F = C * 9/5 + 32
b temp $celsius [*] 9
b temp $temp [/] 5
b fahrenheit $temp [+] 32

say $celsius~C is~$fahrenheit~F
